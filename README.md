## 🇺🇸 English Version

**Saku Admin** is a comprehensive multi-platform management solution (Web, Android, and iOS) designed to handle a full e-commerce and services ecosystem. Built with cutting-edge technologies, it provides a premium, fluid user experience for real-time, data-driven decision-making.

### 🌟 Key Features
- **Premium Analytical Dashboard**: Real-time sales visualization (Daily, Weekly, Monthly, Yearly) with custom SVG charts.
- **Inventory & Catalog Management**: Full control over products and categories with image upload support.
- **Orders & Logistics Control**: Detailed order tracking, dynamic statuses, and shipping configuration.
- **User Management & Notifications**: Direct communication via segmentated **Push Notifications** and integrated scheduling.

### 🛠️ Tech Stack
- **Framework**: [Expo](https://expo.dev/) (SDK 55) & [React Native](https://reactnative.dev/).
- **Navigation**: [Expo Router](https://docs.expo.dev/router/introduction/).
- **Backend**: [Firebase Firestore](https://firebase.google.com/docs/firestore) & [Authentication](https://firebase.google.com/docs/auth).
- **Styling**: [NativeWind](https://www.nativewind.dev/) (Tailwind CSS) & [Reanimated](https://docs.swmansion.com/react-native-reanimated/).
- **Maps**: [React Google Maps API](https://visgl.github.io/react-google-maps/).

## 🇪🇸 Versión en Español

**Saku Admin** es una solución integral de gestión multiplataforma (Web, Android e iOS) diseñada para administrar un ecosistema completo de e-commerce y servicios. Construida con tecnologías de vanguardia, ofrece una experiencia de usuario premium y fluida para la toma de decisiones basada en datos en tiempo real.

### 🌟 Características Principales
- **Dashboard Analítico Premium**: Visualización de ventas en tiempo real (Diario, Semanal, Mensual, Anual) con gráficos SVG personalizados.
- **Gestión de Inventario y Catálogo**: Control total sobre productos y categorías con soporte para carga de imágenes.
- **Control de Pedidos y Logística**: Seguimiento detallado de órdenes, estados dinámicos y configuración de envíos.
- **Gestión de Usuarios y Notificaciones**: Comunicación directa mediante **Notificaciones Push** segmentadas y agenda integrada.

### 🛠️ Stack Tecnológico
- **Framework**: [Expo](https://expo.dev/) (SDK 55) y [React Native](https://reactnative.dev/).
- **Navegación**: [Expo Router](https://docs.expo.dev/router/introduction/).
- **Backend**: [Firebase Firestore](https://firebase.google.com/docs/firestore) y [Authentication](https://firebase.google.com/docs/auth).
- **Estilos**: [NativeWind](https://www.nativewind.dev/) (Tailwind CSS) y [Reanimated](https://docs.swmansion.com/react-native-reanimated/).
- **Maps**: [React Google Maps API](https://visgl.github.io/react-google-maps/).

## 🚀 Setup & Installation / Instalación

1. **Clone the repo / Clonar repo**:
   ```bash
   git clone https://github.com/trabbittbusiness-spec/saku-admin-tienda.git
   cd saku-admin-tienda
   ```

2. **Install dependencies / Instalar dependencias**:
   ```bash
   npm install
   ```

3. **Run the project / Iniciar proyecto**:
   ```bash
   npx expo start
   ```
